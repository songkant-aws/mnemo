# Review Memory Quality

Use this skill when the user wants to clean up or improve memory quality.

## Commands

```bash
~/plugins/mnemo/scripts/mnemo merge
~/plugins/mnemo/scripts/mnemo feedback
```

## Guidance

- Use merge for duplicate names that represent the same thing.
- Use feedback to improve summaries, remove junk, or fix weak relations.
- Prefer replayable merge/feedback events over hand-editing generated views.
